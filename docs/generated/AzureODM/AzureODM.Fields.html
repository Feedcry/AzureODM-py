class AzureODM.Fields.JSONField(required=False, keep_Null=False)[source]¶

Bases: AzureODM.Fields.GenericField

This is a field that store content in JSON

The type of this field is a Python Object (List, or Dict), when store it into the database, it need be converted into JSON by calling serialize(). When retrieve it, it will be converted back into Python Object by calling: deserialize()
